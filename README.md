# JS Assignment - Fundamentals of Web Design

## 📌 Overview
This repository contains solutions to 6 JavaScript problems using basic concepts like loops, conditions, and functions. Input is taken using prompt() and output is shown using alert().

---

## 🔹 Question 1: Digit Gatekeeper
**Approach:**
- Loop from L to R
- Check divisibility by K
- Skip numbers containing 0
- Find sum of digits and check if prime

**Time Complexity:** O(N * d)

---

## 🔹 Question 2: Roll-Seed Lock
**Approach:**
- Perform transformation 3 times
- Check final number range (100–999)
- Verify middle digit

**Time Complexity:** O(1)

---

## 🔹 Question 3: Mirror Corridor
**Approach:**
- Try values of X from 0 to 100000
- Check palindrome and divisibility

**Time Complexity:** O(N)

---

## 🔹 Question 4: Fare Calculator
**Approach:**
- Apply conditions step-by-step
- Use Math.floor and Math.ceil

**Time Complexity:** O(1)

---

## 🔹 Question 5: Skipping Numbers
**Approach:**
- Increment m until sum ≥ N
- Skip multiples of (seed + 2)

**Time Complexity:** O(N)

---

## 🔹 Question 6: Contest Score Judge
**Approach:**
- Calculate score
- Apply conditions
- Determine PASS/FAIL

**Time Complexity:** O(1)
